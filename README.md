# ARCL KiCAD Libraries
KiCAD libraries for ARCL parts

# Installation
## Symbol Library
Add as a Global Library in
```
KiCAD > Preferences > Manage Symbol Libraries 
```

## Footprint Library
Add as a Global Library in
```
KiCAD > Preferences > Manage Footprint Libraries 
```
